🖥️ CodingBoyz Debian Terminal
Web-based Debian terminal with Pterodactyl-style UI.

DebianNodePortLicense

Features
✅ Full bash terminal in browser
✅ Pterodactyl-style dark UI
✅ CodingBoyz branding
✅ Kill Process (Ctrl+C) button
✅ Reboot Core Workspace button
✅ Real-time WebSocket connection
✅ Auto-reconnect on disconnect
✅ System resource monitoring
✅ Responsive design
✅ Suga app importable
# Deploy on Suga
Push this repo to GitHub
Connect GitHub to Suga
Click Import on this repository
Suga will auto-detect the .suga/config.json
App runs on port 8080
Manual Deploy
docker-compose up -d --build
Access: http://localhost:8080
